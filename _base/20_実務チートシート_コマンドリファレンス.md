# まず確認する5コマンド

```bash
git status

rg -n "対象キーワード" .
```

---

# HTTPステータス早見表

|Code|意味|まず確認すること|
|---:|---|---|
|200|成功|レスポンス内容|
|301|恒久リダイレクト|Location|
|302|一時リダイレクト|Cookie・Session|
|400|リクエスト不正|Payload|
|401|認証|Authorization|
|403|権限|認証・権限|
|404|見つからない|URL・Route・Rewrite|
|405|Method不正|GET/POST|
|500|サーバーエラー|PHPログ|
|502|Bad Gateway|PHP-FPM|
|503|サービス停止|Docker・Webサーバー|

# curl チートシート

```bash

curl -L URL
curl -o out.html URL
curl -X POST -d "a=1&b=2" URL
curl -H "Content-Type: application/json" -d '{"id":1}' URL
curl -s -o /dev/null -w "%{http_code} %{time_total}\n" URL
```
