# CLI・デバッグ実践ハンドブック

> 「勘で直す」から卒業する。

この教材は、PHP・Docker・WordPress案件を題材に、

- CLI
- HTTP
- Docker
- SQL
- Git
- VS Code Debug
- Xdebug

を使って

**「証拠を集めて原因を特定する力」**

を身につけるための教材です。

---

# この教材の特徴

普通の参考書ではありません。

本書は

「実務でバグを直す」

ことだけを目的にしています。

説明を読むだけではなく

毎日30〜45分

実際にコマンドを打ちながら学習してください。

---

# 学習目標

最終的に以下が30分以内でできるようになります。

✅ バグを再現する

✅ コード位置を特定する

✅ curlだけでAPIを再現する

✅ Dockerコンテナを調査する

✅ SQLで事実確認する

✅ Xdebugで変数を追う

✅ Gitで原因コミットを探す

---

# 前提知識

本書は

- PHP
- HTML
- Docker

を少し触ったことがある人を対象としています。

Linux経験は不要です。

---

# 推奨環境

VS Code
