# CONTRIBUTING.md

## 開発方針

この教材は、まず全体を作成し、その後に学習しながら改善していきます。

## コミット単位

推奨コミット単位:

- `Add project guidelines`
- `Add module01 cli lessons`
- `Add module02 linux lessons`

## ブランチ

個人利用の場合は `main` へ直接コミットで問題ありません。

将来的に整理する場合:

- `main`
- `feature/module01-cli`
- `feature/module02-linux`

## 更新時のルール

教材を更新する場合は、以下を確認します。

- Lessonの最低品質ラインを満たしているか
- コマンドがコピペ可能か
- 危険な操作に注意書きがあるか
- 関連リンクがあるか
- 学習ログがあるか
