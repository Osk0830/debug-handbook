# Debug Handbook

> 実務で使えるデバッグ力を身につけるためのハンズオン教材

## コンセプト

- 実務ベース
- 手を動かす
- 推測ではなく証拠で判断する
- PHP / WordPress / React / Docker を中心に学ぶ
- 学習しながら育てる

## 学習フロー

```text
Learn → Try → Break → Debug → Review → Apply
```

- よく使うコマンドはReferenceへ追記
- 迷った判断手順はDebug Flowへ追記
