# PROJECT.md

## プロジェクト名

Debug Handbook

## 目的

この教材は、PHP / WordPress / React / Docker / SQL / Git / Xdebug を中心に、実務で使えるデバッグ力を身につけるためのハンズオン教材です。

## 基本方針

- 推測ではなく、証拠で判断する。
- 読むだけでなく、必ず手を動かす。
- コマンド集ではなく、調査の思考プロセスを学ぶ。
- 実務で遭遇する障害を教材化する。
- 最初から完璧を目指さず、学習しながら育てる。

## 想定読者

- PHP / WordPress / React / Docker を実務で扱うエンジニア
- CLIやデバッグに苦手意識がある人
- 「なんとなく直す」から卒業したい人

## 文章トーン

- 日本語
- です・ます調
- 実務寄り
- 難しい言葉は避ける
- ただし必要な専門用語は逃げずに説明する

## Lessonの最低品質ライン

各Lessonには原則として以下を含めます。

- ゴール
- 前提知識
- 実務ではいつ使うか
- 基本
- よくある失敗
- ハンズオン
- 演習
- 模範解答
- FAQ
- 関連リンク
- 学習ログ
- 次へ進む条件

## 優先順位

1. まず最後までデータを揃える
2. 学習しながら不足分を加筆する
3. 実案件で遭遇した内容を追加する
4. MkDocs / PDF / Web版へ展開する
