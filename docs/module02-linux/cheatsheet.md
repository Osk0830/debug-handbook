# Module02 Linux チートシート

```bash
pwd
ls -lah
mkdir sample
mkdir -p sample/src
touch index.php
cp file.txt backup.txt
cp -r src backup/
mv old.txt new.txt
rm file.txt
rm -r dir
cat file.txt
less file.txt
head -20 file.txt
tail -100 app.log
tail -f app.log
chmod 644 file.php
chmod 755 script.sh
ps aux
top
kill <PID>
```
