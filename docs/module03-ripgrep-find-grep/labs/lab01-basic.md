# Module03 ripgrep・grep・find Lab01 Basic

## 目的

大量のファイルから目的のコード・設定・ログを素早く見つけられるようになる。

## 手順

1. Module READMEを読む
2. Lessonを1つ選ぶ
3. そのLessonのハンズオンを実行する
4. 結果をメモする

## 記録テンプレート

```markdown
## 実行したこと

## 結果

## 分かったこと

## 次に確認すること
```

## 完了条件

- [ ] 1つ以上コマンドまたは確認作業を実行した
- [ ] 結果をメモした
- [ ] 次に確認することを書いた
