# Lab01 障害調査コマンド実践

## シナリオ

`http://localhost:8081/wedding/`が500を返しています。Docker Compose環境でPHPとMySQLを使用しています。

## 課題

1. 現在のGitコミットを保存する
2. Dockerサービスの状態を確認する
3. Web・PHP・DBログを確認する
4. curlでレスポンスヘッダーとボディを保存する
5. PHPの構文エラーを検索する
6. DB接続と対象テーブルを確認する
7. 原因、修正、再確認結果を記録する

## 記録テンプレート

```text
実行コマンド:
結果:
分かったこと:
次のコマンド:
```

## 完了条件

- 証拠を残してから変更している
- 各レイヤーの入力と出力を確認している
- 修正後にcurlとブラウザの両方で再確認している
