# Lab02 VS Code総合デバッグワークフロー

## 目的

検索、Problems、Terminal、Debugger、Git履歴を順番に使い、推測ではなく証拠から原因へ到達します。

## シナリオ

APIから取得したユーザー一覧で、特定ユーザーだけ詳細画面へ遷移できません。Consoleには `Cannot read properties of undefined` が出ています。

## 手順

1. エラー全文、URL、対象ID、発生時刻を記録する
2. Problemsを確認し、静的診断の有無を記録する
3. エラーメッセージと関連プロパティをWorkspace検索する
4. Outlineと参照検索でデータ変換処理を追う
5. Terminalでテストまたは型チェックを実行する
6. 条件付きブレークポイントで対象IDだけ停止する
7. Call StackとVariablesから値が消える境界を特定する
8. Git履歴で直近変更の目的を確認する
9. 最小修正と回帰テストを追加する
10. Git差分から調査用コードが残っていないか確認する

## 成果物

- 再現手順
- 確認済み事実
- 原因仮説と検証結果
- 根本原因
- 修正内容
- 回帰確認
- 再発防止

## 合格基準

- 「たぶん」ではなくログ・変数・差分を根拠に説明している
- 正常データと異常データを比較している
- 修正範囲が原因に対して最小である
- テストまたは再現手順で修正を確認している
