# Module21 VS Code チートシート

## 検索

```text
Cmd + P        ファイル名検索
Cmd + Shift + F プロジェクト全体検索
Cmd + Shift + O ファイル内シンボル検索
Cmd + T        ワークスペースシンボル検索
```

## デバッグ

```text
F5             開始
F10            Step Over
F11            Step Into
Shift + F11    Step Out
```

## CLI連携

```bash
rg -n "keyword" .
git status
git diff
docker compose logs -f web-php8
curl -I http://localhost:8081/
```
