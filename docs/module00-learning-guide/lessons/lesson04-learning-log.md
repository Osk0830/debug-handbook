# Lesson04 学習ログの残し方

## ゴール

学習内容・疑問・実務への応用を後から見返せる形で残せるようになります。

## 最小テンプレート

```markdown
# YYYY-MM-DD Lesson名

## 実行したこと

## 結果

## 分かったこと

## 分からなかったこと

## 次に試すこと
```

## 良いログ

良いログは、コマンドだけでなく判断も残します。

```markdown
## 実行したこと

`curl -I` でレスポンスヘッダーを確認した。

## 結果

302でLocationが `/login` だった。

## 分かったこと

アプリの表示エラーではなく、認証判定でログインへ転送されている。

## 次に試すこと

Cookieを保存して同じURLを再実行する。
```

## 次へ進む条件

- [ ] 実行結果と解釈を分けて書ける
- [ ] 「次に確認すること」を書ける
