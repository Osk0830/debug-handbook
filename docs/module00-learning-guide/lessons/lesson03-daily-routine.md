# Lesson03 毎日の学習ルーティン

## 30〜45分の基本形

### 5分: 前回の復習

- 前回の学習ログを読む
- 前回のコマンドを1つ実行する

### 15分: Lessonを読む

- ゴールを確認する
- コマンドの目的を理解する

### 15分: ハンズオン

- 実際に実行する
- 出力を保存する
- 条件を1つ変える

### 5〜10分: 記録

- 分かったこと
- 分からなかったこと
- 次回やること

## 推奨ルール

- 1日に詰め込みすぎない
- 分からない用語は放置しない
- コマンドを暗記しようとしない
- 出力の意味を説明する
- 実案件と結び付ける

## 次へ進む条件

- [ ] 自分の学習時間帯を決めた
- [ ] 30〜45分の進め方を説明できる
