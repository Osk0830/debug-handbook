# Lesson01 この教材の読み方

## ゴール

この教材を、読むだけの資料ではなく、実務力を育てるハンズオン教材として使えるようになります。

## 1Lessonの進め方

1. ゴールを読む
2. 実務で使う場面を確認する
3. コマンド・設定を試す
4. 出力を読む
5. 演習を解く
6. 疑問点を書く
7. 教材へ追記する

## 読むだけで終わらせない

悪い学習:

```text
コマンドを見る
  ↓
分かった気になる
  ↓
数日後に忘れる
```

推奨する学習:

```text
コマンドを見る
  ↓
実行する
  ↓
出力を説明する
  ↓
条件を変えて再実行する
  ↓
実務で使う
```
