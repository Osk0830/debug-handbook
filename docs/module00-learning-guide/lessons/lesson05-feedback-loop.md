# Lesson05 実案件を教材へ戻す方法

## ゴール

仕事で遭遇した問題を、再利用できる教材へ変換できるようになります。

## 実案件から残すもの

- 現象
- 再現手順
- 調査順
- 使ったコマンド
- 決め手になった証拠
- 原因
- 修正
- 再確認
- 再発防止

## 追記先の判断

|内容|追記先|
|---|---|
|用語や仕組み|該当Lesson|
|短いコマンド|cheatsheet.md|
|よくある質問|faq.md|
|症状別対応|troubleshooting.md|
|複合的な障害|Module18 Case Study|
|反復問題|Module19 / Module25|

## 匿名化

次は必ず伏せます。

- 顧客名
- 案件名
- 本番URL
- IPアドレス
- メールアドレス
- token
- password
- DB接続情報

## 次へ進む条件

- [ ] 実案件を教材化するテンプレートを説明できる
- [ ] 追記先を判断できる
