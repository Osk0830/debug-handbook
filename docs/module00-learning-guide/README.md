# Module00 学習ガイド

## このModuleの目的

Debug Handbookの使い方、学習順序、学習記録、環境準備を理解します。

Module01へ進む前に、最低限このREADMEとLesson01を読んでください。

## この教材が目指すもの

この教材の目的は、コマンドを暗記することではありません。

```text
現象を整理する
  ↓
証拠を集める
  ↓
仮説を立てる
  ↓
検証する
  ↓
最小限で修正する
  ↓
同じ手順で再確認する
```

この流れを、CLI・HTTP・Docker・SQL・Git・VS Codeなどを組み合わせて実行できることを目指します。

## 学習時間の目安

- 平日: 30〜45分
- ハンズオン日: 60〜90分
