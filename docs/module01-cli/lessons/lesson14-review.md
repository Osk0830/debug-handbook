# Lesson14 Module01総復習

## ゴール

CLI基礎を一通り使える状態にする。

## 総復習コマンド

```bash
pwd
ls -lah
cd ..
cd -
echo $SHELL
echo $PATH
which php
type cd
history
```

Docker:

```bash
docker compose ps
docker compose exec web-php8 bash
pwd
ls -lah
php -v
exit
```
