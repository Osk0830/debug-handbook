# Lesson11 履歴と補完

## ゴール

コマンド履歴とTab補完を使って、入力ミスと作業時間を減らす。

## 基本

```bash
history
```

履歴検索:

```text
Ctrl + r
```

Tab補完:

```text
途中まで入力して Tab
```

## 実務ではいつ使う？

長いDockerコマンドやcurlコマンドを再利用するとき。

## ハンズオン

```bash
history
```

その後 `Ctrl + r` で過去の `docker` コマンドを探します。

## 演習

1. 過去のコマンドを検索するショートカットは？
2. Tab補完のメリットは？

## 模範解答

1. `Ctrl + r`
2. 入力ミスが減る。速い。
