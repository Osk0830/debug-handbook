# Lesson05 cd

## ゴール

ディレクトリを迷わず移動できるようになる。

## 基本

```bash
cd docs
cd ..
cd ../..
cd ~
cd -
```

## 実務ではいつ使う？

- プロジェクトルートへ移動する
- ログディレクトリへ移動する
- Dockerコンテナ内で作業場所を移動する

## 重要: cd -

直前にいた場所へ戻ります。

```bash
cd docs
cd -
```

## よくある失敗

深い階層から戻るために `../../../../` を何度も手打ちする。

`cd -` やタブ補完を使いましょう。

## ハンズオン

```bash
pwd
cd ..
pwd
cd -
pwd
```

## 演習

1. `cd -` は何をしますか？
2. ホームディレクトリに移動するコマンドは？

## 模範解答

1. 直前のディレクトリへ戻る。
2. `cd ~`
