# Lesson04 ls

## ゴール

ファイルとディレクトリの一覧を確認できるようになる。

## 実務ではいつ使う？

- 設定ファイルの有無確認
- ログファイルの存在確認
- 権限や更新日時の確認
- Dockerコンテナ内のファイル確認

## 基本

```bash
ls
ls -l
ls -la
ls -lah
```

一番よく使うのはこれです。

```bash
ls -lah
```

## オプション

|オプション|意味|
|---|---|
|`-l`|詳細表示|
|`-a`|隠しファイルも表示|
|`-h`|サイズを読みやすく表示|

## よくある失敗

`.env` や `.gitignore` は隠しファイルなので、`ls` だけでは表示されません。

```bash
ls -la
```

を使います。

## ハンズオン

```bash
ls
ls -lah
ls -la
```

## 演習

1. `.env` が表示されない時に使うコマンドは？
2. ファイルサイズを読みやすく表示するオプションは？

## 模範解答

1. `ls -la`
2. `-h`

## FAQ

### Q. `ls -lah` は毎回使っていいですか？

はい。実務ではかなりよく使います。
