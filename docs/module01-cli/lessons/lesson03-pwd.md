# Lesson03 pwd

## ゴール

現在地を確認する癖をつける。

## 実務ではいつ使う？

- Git操作前
- `rm` や `mv` など危険な操作前
- Dockerコンテナ内で作業場所を確認するとき
- ログや設定ファイルを探すとき

## 基本

```bash
pwd
```

`pwd` は **print working directory** の略です。

現在いるディレクトリを表示します。

## なぜ重要？

CLIでは、今いる場所によってコマンドの結果が変わります。

```bash
ls
```

も、どこで実行するかによって表示される内容が違います。

## よくある失敗

違うディレクトリで `git status` を実行して、別リポジトリを見てしまう。

## ハンズオン

```bash
pwd
cd ..
pwd
cd -
pwd
```

## 演習

1. `pwd` は何の略ですか？
2. Git操作前に `pwd` を確認する理由は何ですか？

## 模範解答

1. print working directory
2. 違うディレクトリで操作する事故を防ぐため。

## FAQ

### Q. VS Codeで開いているフォルダとpwdが違うことはありますか？

あります。統合ターミナルで別の場所に移動している場合があります。
