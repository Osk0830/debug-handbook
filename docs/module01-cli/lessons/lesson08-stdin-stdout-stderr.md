# Lesson08 標準入力・標準出力・標準エラー

## ゴール

CLIの入出力の基本を理解する。

## 基本

|名前|番号|意味|
|---|---:|---|
|stdin|0|標準入力|
|stdout|1|標準出力|
|stderr|2|標準エラー|

## 実務ではいつ使う？

- ログ保存
- エラーだけ別ファイルに保存
- コマンド結果を別コマンドへ渡す

## ハンズオン

```bash
ls
ls not-found
```

エラーだけ保存:

```bash
ls not-found 2> error.log
cat error.log
```

## 演習

1. stdoutとstderrの違いは？
2. stderrの番号は？

## 模範解答

1. 通常の結果とエラー出力。
2. 2。
